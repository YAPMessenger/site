# site

The public marketing website for YAP — **yap-messenger.com**, served via GitHub Pages from this repo (`main`).

## Layout
- `index.html` · `privacy.html` · `terms.html` · `faq.html` — the pages.
- `CNAME` — custom domain for GitHub Pages.
- `docs/` — infrastructure + ops reference (e.g. `Website-Infrastructure.md`). Flat, since the site is small.

## Deploy
Push to `main`; GitHub Pages publishes within a few minutes.

## Conventions
Follows the shared YAP repo conventions — see **`docs/CONVENTIONS.md`**. (Public repo → commits are
authored as `YAP Messenger <contact@yap-messenger.com>`.)
